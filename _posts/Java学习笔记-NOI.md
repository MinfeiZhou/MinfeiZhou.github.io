## 0 基本概念

方法引用

借助Comparator.comparing工厂方法，运用「方法引用」，不仅得到一个没有重复的设计，而且进一步提高了代码的可读性。「方法引用」其本质是具有单一方法调用的lambda表达式的「语法糖」表示。

import static java.util.Comparator.comparing;

repo.sort(comparing(Product::getWeight));
